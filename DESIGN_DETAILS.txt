🎨 STRUCTURE VISUELLE
═════════════════════════════════════════════════════════════════════════════════

La page login a cette structure:

├─ <html lang="fr">
│  ├─ <head>
│  │  ├─ Métadonnées
│  │  ├─ Bootstrap CSS (CDN)
│  │  └─ CSS Personnalisé (80 lignes)
│  │
│  └─ <body>
│     ├─ Background: Dégradé violet/bleu full-screen
│     ├─ Flexbox: Centrage vertical + horizontal
│     │
│     └─ <div class="login-container">  (max-width: 400px)
│        ├─ Alerte erreur (si erreur)
│        ├─ Alerte succès (si succès)
│        │
│        └─ <div class="card login-card">
│           ├─ <div class="login-header">
│           │  ├─ <h1>Connexion</h1>
│           │  └─ <p>Accédez à votre espace sécurisé</p>
│           │
│           └─ <div class="login-body">
│              ├─ <form method="POST" action="...">
│              │  ├─ <div class="form-group">
│              │  │  ├─ <label for="email">Adresse email</label>
│              │  │  └─ <input type="email" ... />
│              │  │
│              │  ├─ <div class="form-group">
│              │  │  ├─ <label for="password">Mot de passe</label>
│              │  │  └─ <input type="password" ... />
│              │  │
│              │  └─ <button type="submit" class="btn-login">
│              │     Se connecter
│              │     </button>


🎯 PALETTES DE COULEURS
═════════════════════════════════════════════════════════════════════════════════

Dégradé Principal (Background):
  Couleur 1: #667eea (Bleu violet clair)
  Couleur 2: #764ba2 (Violet foncé)
  Direction: 135deg (angle diagonal)
  Résultat: Gradient élégant et dynamique

Textes:
  Primary:   #2c3e50 (Bleu gris foncé) - Titres, labels
  Secondary: #6c757d (Gris moyen) - Sous-titres
  White:     #ffffff (Blanc pur) - Textes clairs

Bordures et Focus:
  Border:    #e9ecef (Gris très clair)
  Focus:     #3498db (Bleu ciel) - Quand on clique
  Ombre Focus: rgba(52, 152, 219, 0.1) - Ombre légère bleue

Alertes:
  Erreur: Fond #fee, Texte #c33 (rouge léger)
  Succès: Fond #efe, Texte #3c3 (vert léger)

💫 EFFET AU SURVOL/CLIC
═════════════════════════════════════════════════════════════════════════════════

Input Email/Password au Focus:
  ┌─────────────────────┐
  │ border: 2px #3498db │ (Bleu ciel)
  │ outline: none       │ (Pas de contour par défaut)
  │ box-shadow: inset   │ (Ombre interne bleue)
  └─────────────────────┘

Bouton "Se connecter" au Survol:
  ┌─────────────────────┐
  │ transform: translateY(-2px)  (Remonte légèrement)
  │ box-shadow: plus prononcée   (Ombre augmentée)
  └─────────────────────┘

Bouton "Se connecter" au Clic:
  ┌─────────────────────┐
  │ transform: translateY(0)     (Revient à la place)
  └─────────────────────┘

═════════════════════════════════════════════════════════════════════════════════

Affichage des erreurs:
  Classe: alert alert-danger

  Texte: #c33 (rouge)
  Padding: 12px 15px

       |← Bouton fermer (x)

Affichage du succès:
  Classe: alert alert-success
