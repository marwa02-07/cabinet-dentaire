⚠️ SI UN TEST ÉCHOUE
═════════════════════════════════════════════════════════════════════════════════

Si vous voyez du debug affiché:
  → Vérifiez reinit.php:
     http://localhost/web-hopital/reinit.php
  → Rafraîchissez la page (Ctrl+F5)
  → Essayez incognito (Ctrl+Shift+N)

Si la page ne charge pas:
  → Vérifiez que MySQL est actif
  → Vérifiez les fichiers existent
  → Vérifiez l'URL exacte

Si les erreurs ne s'affichent pas:
  → Vérifiez browser console (F12)
  → Vérifiez les logs Apache
  → Vérifiez les sessions PHP
